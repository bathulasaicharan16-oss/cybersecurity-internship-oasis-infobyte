# Task 1 – Basic Network Scanning with Nmap

## Objective
Perform a network scan using Nmap to identify open ports and services.

## Tool Used
Nmap

## Steps Performed
1. Installed Nmap on Kali Linux
2. Identified local IP address using "ip address"
3. Performed scan using command:
   nmap 10.0.2.15
4. Analyzed the scan results

## Result
The scan shows the host is active but all scanned ports are filtered. 
This indicates firewall protection or restricted network access.

## Conclusion
Nmap helps security professionals identify open ports and services 
running on a system which helps detect potential vulnerabilities.
